• A hyperlink (or link) is a digital reference that connects one web page to another web document, file, or specific location within a page. Users click or tap a hyperlink to navigate from one place on the internet to another. Hyperlinks are created in HTML using the {a} (anchor) tag with an href (hypertext reference) attribute that holds the target URL.
