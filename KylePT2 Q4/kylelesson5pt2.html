‧₊˚ ┊ Selectors, Div, Span, Class, IDs, and Links

• CSS selectors are used to choose which HTML elements will be styled on a webpage. These can include element selectors like p or h1, class selectors written with a dot (e.g., .box), and ID selectors written with a hashtag (e.g., #header). The {div} tag is a block-level container that groups sections of content for layout and styling, while the {span} tag is an inline container used to style small parts of text within a line. A class can be applied to multiple elements to give them the same style, whereas an ID is unique and should only be used once per page. The {a} (anchor) tag creates hyperlinks, and CSS can style links by changing their color, removing underlines, and adding effects like hover transitions.
